# NotesApp :memo:

- A Node.js CLI application for management of notes.

![Alt Text](https://media.giphy.com/media/jcDFbFo7FYDjGn6jcS/giphy.gif)

## Idea :bulb:

- To create a Node.js CLI application for managing notes

## Usage

### Help
```
node app.js --help
app.js [command]

Commands:
  app.js add     Add a new note
  app.js remove  Remove a note
  app.js list    List notes
  app.js read    Read a note

Options:
  --help     Show help                                                 [boolean]
  --version  Show version number                                       [boolean]
```
### Create Note
```
node app.js add --title="MyTitle" --body="Sample Body"
```

## Features :boom:

- :heavy_check_mark: Save your notes to notes.json file in the root directory of the project
- :heavy_check_mark: Create, Read, Remove, List Notes
- :heavy_check_mark: Colors for CLI Output
